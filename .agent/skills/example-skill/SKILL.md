# Example Skill

This is a template skill based on the documentation.

## Structure
- `SKILL.md`: This file, containing instructions.
- `scripts/`: Optional directory for helper scripts.
- `examples/`: Optional directory for examples.
- `resources/`: Optional directory for resources.

## Usage
When this skill is relevant, the agent will read this file and follow the instructions.
